# Crypto Chat Lab - Algoritmo de César

Este es un proyecto sencillo que demuestra cómo funciona la criptografía utilizando el **Algoritmo de César**. El proyecto muestra dos chats en una sola pantalla, donde puedes observar cómo el texto se encripta y desencripta usando este algoritmo.

## Descripción

El proyecto tiene dos interfaces de chat en una sola pantalla. El primer chat recibe el texto en su forma original, mientras que el segundo chat muestra el mismo texto cifrado o descifrado, dependiendo de la operación seleccionada. El algoritmo de César es un método de cifrado por sustitución en el que cada letra del texto se reemplaza por otra que se encuentra un número fijo de posiciones más adelante en el alfabeto.

## Estructura del Proyecto

El proyecto contiene los siguientes archivos y directorios:

```

...\crypto-chat-lab
│
├── .vscode/            # Configuración de Visual Studio Code (opcional)
│
├── src/                # Archivos de código fuente (JavaScript y HTML)
│
└── index.html          # Página principal que se ejecuta en el navegador

```

## Cómo Ejecutarlo

Este proyecto no depende de frameworks ni bibliotecas externas. Solo necesitas abrir el archivo `index.html` en tu navegador para ver la demostración del algoritmo de César en acción.

1. Navega al directorio del proyecto.
2. Abre el archivo `index.html` en cualquier navegador web moderno.
3. Interactúa con los chats para ver cómo el texto se encripta y desencripta.

## Funcionalidad

- **Encriptación:** Cuando escribes un mensaje en el chat original, este se encripta automáticamente usando el algoritmo de César y se muestra en el chat cifrado.
- **Desencriptación:** Al hacer clic en el chat cifrado, el mensaje se descifra y muestra el texto original.
  
El algoritmo es simple y permite desplazar las letras de un mensaje por un número fijo de posiciones.
